## 二、每个改了什么

每处改造严格对照 reference 模板：

```
background: linear-gradient(135deg, #4F46E5 0%, #7C3AED 100%)
border: none
box-shadow: 0 4px 12px rgba(79, 70, 229, 0.25)   // 单层 subtle
hover: translateY(-1px) + shadow → 0 6px 16px rgba(79, 70, 229, 0.3)
```

立体特征全部去除：
- ❌ 全部去除 `inset 0 1px 0 rgba(255,255,255,0.18)`（白色高光内边）
- ❌ 全部去除多层 shadow（`0 1px 2px ..., inset ...` 双层）
- ❌ 全部去除 `linear-gradient(180deg, 三段紫色)` 顶亮底深的凸起渐变
- ❌ AgentPostModal modal 容器 `inset` 改成 `border` 1px

具体改造点：
1. **HomePage 发新帖 CTA** — 这是用户原话指名要改的按钮，从 180deg 三段渐变 + inset 白光 → 135deg 双段紫渐变 + 单层 shadow + hover translateY
2. **Sidebar 未登录态 CTA** — 同上模板
3. **RightPanel Connect 按钮** — 同上模板
4. **CreatePostPage** — 黑色 `bg-[#141414]` 全部换 brand 紫渐变（让 Agent 起草、查看帖子、再发一条、下一步、发布帖子）；Step indicator 当前步圆点也用渐变；分类卡选中态 border 黑→紫
5. **AgentSetupPage** — 下一步 + 创建 Agent 按钮统一到 reference shadow 0.25
6. **AgentSetupDonePage** — 去发第一篇帖子按钮 solid → 渐变
7. **AgentPostModal** — 容器 inset 高光改成 border
8. **UnsubscribePage** — 确认退订 + 返回 Bulletin 按钮 solid → 渐变

## PostCard / CategoryBadge 多色策略

- **PostCard 左 4px 边框**：按 category 微染色（jobs/projects 紫系、marketplace 橙、skills 绿、housing 青、events 粉、all 灰），非 hover 时 opacity 0.55，hover 拉满到 1.0
- **PostCard hover**：背景从纯白改成 `linear-gradient(145deg, #ffffff, accent.glow)` —— category color 的 4% 极弱底，原有逻辑保留
- **CategoryBadge**：保留原有 7 种 category 配色（已经是浅色 + 深色文字 + 同色 dot），与新色板兼容；未单独改动
- **Sort tab active**：从白底改浅紫底 + 紫色文字 + 紫色 inset border，与"亮色拼接"主题统一

## Hover / 微互动

- Sidebar 菜单项 hover：浅紫 `#F5F3FF`；active：浅紫 `#EDE9FE` + 紫色 2px 左边框
- PostCard hover：translateY(-1px) + 阴影加深 + 边框变 category color 40% 透明（原有逻辑保留）
- RightPanel trending tags：白底 hover 浅紫
- Hero CTA "让我的 Agent 发帖"：白底紫字 → hover 浅紫底 + 阴影加深

## 字体

`body` font-family 改为 `'PingFang SC', 'Inter', -apple-system, ...`，中文优先。

## a11y / contrast

- 主文字 `#1A1A1E` 在所有亮底（白 / 米 / 浅紫 / 浅蓝 / 浅黄 / 浅绿）上 contrast ratio ≥ 12:1（远超 AAA 7:1）
- 次文字 `#6B6B66` 在白底上约 5.7:1（AA Large 通过；AA Normal 4.5:1 通过）
- 紫色 `#4F46E5` 在所有浅底上 contrast ≥ 7:1
- 暖黄分隔条文字 `#92400E` 在 `#FFFBEB` 上 ≥ 8:1
- 未发现 contrast 问题

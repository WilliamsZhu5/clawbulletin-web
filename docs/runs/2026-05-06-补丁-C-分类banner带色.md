# 补丁 C — 分类页 hero banner 按 category 主色

日期：2026-05-06
范围：HomePage 顶部分类 banner（`/c/:category` 路由）

## 用户原话
"分类当中，所有的分类比如说 housing 这种，上面的主题框都像上面的一样带有颜色"

## 实施方式

在 `HomePage.tsx` 内引入 `分类Banner主题: Record<CategoryId, ...>` 配置表，
按当前 `activeCategory`（来自 `useParams<{ category }>()` → `/c/:category`）切换：
- gradient 背景（白 → 浅色 → 深一档浅色）
- border 同色弱化
- 圆角方块 icon 容器（lucide icon + 浅色底 + category 主色）
- 中文 / 英文 hero 文案 + tagline（按 `useLanguage().lang` 切换）

## 改动文件（1 个）

- `src/app/pages/HomePage.tsx`
  - import 新增 7 个 lucide icon：Briefcase / Rocket / ShoppingBag / Wrench / Building2 / CalendarDays / LayoutGrid
  - 删 `categories` import + 删 `activeCategoryData` 局部变量（原 banner 用它取 label，新 banner 改用主题表）
  - 在 return 前加 `分类Banner主题` 配置表（7 个 category）
  - 替换分类页 banner JSX（原本同色紫蓝 → IIFE 包裹 + 按主题渲染）

## 覆盖的 category 数：7

| category | gradient 终止色 | icon | 中文文案 |
|---|---|---|---|
| jobs | indigo `#E0E7FF` | Briefcase | 招聘 / 求职 — Agent 撮合工作机会 |
| projects | purple `#DDD6FE` | Rocket | 项目协作 — Agent 找队友 / 找项目 |
| marketplace | orange `#FED7AA` | ShoppingBag | 商品市场 — Agent 撮合买卖 |
| skills | green `#BBF7D0` | Wrench | 技能服务 — Agent 提供专业服务 |
| housing | cyan `#A5F3FC` | Building2 | 租房 / 住宿 — Agent 撮合房源 |
| events | rose `#FBCFE8` | CalendarDays | 活动 — Agent 组织 / 参加 |
| all | slate（保留紫蓝渐变） | LayoutGrid | 全部 — Agent 们的公告板 |

## default 'all' 状态

`isHome === true` 分支（即 `/`）保留原紫蓝色 hero 不变（"Welcome back, X"）。
分类页 `activeCategory === 'all'` 极少触发（路由通常进 isHome），但若触发则用 slate 主题兜底。

## 功能保留
- ✅ 没动数据加载、排序、subcategory pills、posts 列表
- ✅ 没动 isHome 分支（紫蓝 banner + Welcome back）
- ✅ 没动 onboarding guide
- ✅ 没动其他页面（Sidebar / TopBar / LoginPage / Notifications / Messages 都没碰）
- ✅ 没动后端
- ✅ 没装新依赖（icons 全在已有 `lucide-react`）

## 颜色复用
全部从已有 `src/styles/theme.css` 的 `--cat-*-bg/text/dot/border` 设计 token 取色，
与 `CategoryBadge` 保持一致；浅 gradient 终止色基于 Tailwind 标准浅档（200 级别）保证文字对比度。

## 移动端
- banner 用 `flex items-start gap-3 sm:gap-4`，icon 11×11 固定，title/desc 区 `flex-1 min-w-0`
- padding `22px 24px 20px` 在窄屏不会撑爆
- 不破坏响应式
